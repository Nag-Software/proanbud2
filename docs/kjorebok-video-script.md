# Kjørebok – videomanus (markedsføring)

**Lengde:** ~45–60 sek · **Format:** skjermopptak (desktop 16:9 + mobil-innslag) · **Tone:** rolig, selvsikker, «dette sparer deg for tid»

---

## Hook (0–5 sek)
| | |
|---|---|
| **Skjerm** | Telefon i bil-holder. Sjåfør trykker **«Start kjøring»**. Kart begynner å spore ruten. |
| **Voiceover** | «Kjørebok på papir? Det er 2026.» |
| **Tekst på skjerm** | KJØREBOK · automatisk |

## Problem → løsning (5–15 sek)
| | |
|---|---|
| **Skjerm** | Klipp: kvitteringer/Excel-ark → bytter til ProAnbud kjørebok-oversikten med rene tall. |
| **Voiceover** | «Glem manuell logging og rot med satser. ProAnbud fører turen for deg – fra du starter til den er klar for regnskapet.» |
| **Tekst på skjerm** | Statens satser · automatisk |

## Demo 1 – Live sporing (15–28 sek)
| | |
|---|---|
| **Skjerm** | «Start kjøring» → live-kart sporer rute. Stopp → skjema fylles automatisk med distanse. |
| **Voiceover** | «Trykk start når du kjører. Vi sporer ruten med GPS. Når du stopper, er distanse, godtgjørelse og drivstoff allerede regnet ut.» |
| **Tekst på skjerm** | GPS-sporing · ingen manuell km |

## Demo 2 – Ny tur / kart (28–40 sek)
| | |
|---|---|
| **Skjerm** | «Ny tur»-siden: kart til venstre, **Fra hvor? → Til hvor?**, skjema til høyre (kjøretøy, prosjekt, sjåfør, formål). Distanse + godtgjørelse + drivstoff oppdateres live nederst. |
| **Voiceover** | «Eller legg inn en tur manuelt. Skriv fra og til – så regner vi ut kilometer, godtgjørelse etter statens satser, og drivstoffkostnad per mil.» |
| **Tekst på skjerm** | Yrke / privat · per prosjekt |

## Demo 3 – Oversikt + Tripletex (40–52 sek)
| | |
|---|---|
| **Skjerm** | Oversikt: statistikk-fliser (distanse, godtgjørelse, drivstoff, antall turer), tabell med turer, faner **Per prosjekt / Per sjåfør / Kjøretøy**. Trykk «overfør» → badge blir **Synket**. |
| **Voiceover** | «Alt samles på ett sted – fordelt på prosjekt og sjåfør. Med ett klikk havner reiseregningen rett i Tripletex.» |
| **Tekst på skjerm** | Rett til Tripletex |

## Avslutning / CTA (52–60 sek)
| | |
|---|---|
| **Skjerm** | Logo + nettadresse. |
| **Voiceover** | «Kjørebok, ferdig ført. Prøv ProAnbud i dag.» |
| **Tekst på skjerm** | proanbud.no · Kom i gang |

---

## Nøkkelpunkter å få med visuelt
- GPS / live-sporing («Start kjøring»)
- Statens satser regnes automatisk
- Drivstoff: liter/mil × kr per liter
- Yrke vs. privat klassifisering
- Fordeling per prosjekt og per sjåfør
- Ett-klikks overføring til Tripletex (badge: Synket)
- Fungerer på mobil i bil (CarPlay-vennlig kart)

## Opptaks-tips
- Seed et par realistiske turer først, så oversikten ikke er tom (0 km gir dårlig inntrykk).
- La kartet på «Ny tur» laste ferdig (vent 3–4 sek) før du filmer – tiles må være synlige.
- Bruk en demo-bedrift med navn/prosjekter som ser ekte ut.
- Rolige, myke scroll-bevegelser; ikke for raskt.
